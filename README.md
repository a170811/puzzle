# 🧩 積木拼圖求解器

一個互動式的拼圖遊戲，支援手動拼圖和自動求解功能。使用 12 種不同形狀的積木，填滿 6x10 的遊戲板。

## 🎮 功能特色

- **手動拼圖模式**：自由選擇和擺放積木
- **自動求解**：AI 算法自動完成剩餘拼圖
- **跨平台支援**：完整支援電腦和手機操作
- **精緻 UI**：木質風格設計，12 種顏色積木
- **即時操作**：旋轉、翻轉積木，找到最佳位置

## 🎯 遊戲玩法

### 電腦操作

1. **選擇積木**：點擊下方積木托盤中的任一積木
2. **旋轉/翻轉**：
   - 按 `R` 鍵：旋轉積木
   - 按 `F` 鍵：翻轉積木
3. **放置積木**：在遊戲板上點擊要放置的位置
4. **移除積木**：右鍵點擊遊戲板上的積木即可移除

### 📱 手機操作

1. **選擇積木**：點擊下方積木托盤中的任一積木
2. **旋轉/翻轉**：使用螢幕上出現的操作按鈕
   - 🔄 **旋轉**：順時針旋轉 90 度
   - 🔁 **翻轉**：水平翻轉積木
   - ✖ **取消選擇**：取消當前選擇
3. **放置積木**：在遊戲板上點擊要放置的位置
4. **移除積木**：長按（0.5 秒）遊戲板上的積木即可移除

## 🤖 自動求解

點擊「**根據現狀完成拼圖**」按鈕，系統會自動計算並填滿剩餘空格。如果當前格局無解，系統會提示調整位置。

## 🚀 快速開始

直接在瀏覽器中打開 `index.html` 檔案即可開始遊戲。

```bash
# 使用瀏覽器打開
open index.html
```

## 💡 遊戲技巧

- 先放置形狀特殊的積木（如 X、T、W），因為它們較難找到合適位置
- 長條形積木（I）通常用於填補邊緣或較長的空隙
- 善用旋轉和翻轉功能，每個積木有多種擺放方式
- 避免留下孤立的小空間，這會導致無法完成拼圖

## 🛠️ 技術說明

- **純前端實現**：HTML + CSS + JavaScript
- **無需依賴**：開箱即用，無需安裝任何套件
- **回溯算法**：使用深度優先搜尋（DFS）求解拼圖
- **響應式設計**：自動適配各種螢幕尺寸
